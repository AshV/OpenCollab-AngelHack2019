Get Started

Direct Calls

A direct call can be setup by any one member hosting with a self-defined uniques channel ID (any string). By using the same channel ID, the other member can join the call. To switch from Voice to Video call and vice versa, switch the camera button during the call.

Broadcast/Webinar

A webinar/broadcast can be hosted by the main speaker. The same channel ID on being used by the other members will enable them to join the call. Turning off the video and mic will create a suitable environment. This can be undone too during the call.

Group Calls

A group call or meetings can be set by any one member hosting the call. Using the same channel ID by other members will connect them to the same call room.
